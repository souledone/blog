Difference Between apt update & apt upgrade in Ubuntu


In this blog we try to explain difference between update and upgrade in ubuntu.

sudo apt update

By running above command the apt package manager will fetch the updatesif present and also adds new ppa (if added to source.list)

sudo apt upgrade

This command actually upgrades the packages present in the source list.
